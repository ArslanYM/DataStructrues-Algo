## Problem- Shuffle String
1. Difficulty- Easy 
2. [QuestionLink](https://leetcode.com/problems/shuffle-string/)
3. Approach - Pointer
4. Language - C++


5.Solution
 
 
       class Solution {
    public:
    string restoreString(string s, vector<int>& indices) {
       string str  = s;
       int len = indices.size();
      for(int i = 0;  i < len;i++) 
      {
         str[indices[i]]=s[i];
      }

       
       return str;

    }
    };

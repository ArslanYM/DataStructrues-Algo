## Problem-  Power of Three
1. Difficulty- Easy 
2. [QuestionLink](https://leetcode.com/problems/power-of-three/)
3. Approach -  Iterative
4. Language - C++


5.Solution
       


            class Solution {
            public:
                  bool isPowerOfFour(int n) {
                        if(n<=0)
                           return false;
                        while(n%3==0) {
                            n/=3;
                        }

                     return  n==1;
    
             }
            };

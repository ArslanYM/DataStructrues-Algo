## Problem- Valid Anagram
1. Difficulty- Easy 
2. [QuestionLink](https://leetcode.com/problems/valid-anagram/)
3. Approach - Sortedf the string using `sort()`
4. Language - C++


5.Solution
 
    class Solution {
    public:
    bool isAnagram(string s, string t) {
        if(s.size()!=t.size()) return false;
        sort(t.begin(),t.end());
        sort(s.begin(),s.end());
        return (s==t);
    }
    };

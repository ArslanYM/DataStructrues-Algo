## Problem- Reverse Linked List

1. Difficulty- Easy
2. [QuestionLink](https://leetcode.com/problems/reverse-linked-list/)
3. Approach - Three pointer 
4. Language - C++

5.Solution
               class Solution {
               public:
                   ListNode* reverseList(ListNode* head) {
                       if(head==NULL) {
                     return head;
                       }
                       //using 3 pointers
                       ListNode* prev = NULL;
                       ListNode* pres = head;
                       ListNode* next = pres->next;

                         while(pres!=NULL) {
                            pres->next=prev;
                            prev=pres;
                            pres=next;
                            if(next!=NULL) {
                             next=next->next;
                             }
                         }
                          return prev;
                     }
                      };
